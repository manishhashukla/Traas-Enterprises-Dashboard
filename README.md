## 📊 Traas Enterprises Dashboard

An interactive and dynamic **Power BI dashboard** designed to analyze business performance, sales trends, and financial insights for Traas Enterprises.

---

## 🚀 Project Overview

This dashboard provides a complete view of business operations by transforming raw data into meaningful insights. It helps stakeholders make **data-driven decisions** through visually rich and interactive reports.

---

## 🛠️ Tech Stack

The Dashboard was built using the following tools and technologies:

* 📊 **Power BI Desktop** – Main data Visualization platform used for report creation.
* 📂 **Power Query** – Data Transformation and cleaning layer for reshaping and preparing the data.
* 🧠 **DAX (Data Analysis Expressions)** – Used for calculated mesures, dynamic visuals.
* 📁 **File Format** – .pbix for development and .png for dashboard previews. 
---

## 📈 Key Features

* 💰 **Total Sales Overview** (₹ 4M+ insights)
* 📊 **GST & Revenue Tracking**
* 📉 **Due vs Received Amount Analysis**
* 🗺️ **State-wise Sales Performance**
* 🧾 **Company-wise Financial Summary**
* 📦 **Service-wise Sales Breakdown**
* 📅 **Monthly & Quarterly Analysis**
* 🎛️ **Interactive Filters (Company & State)**

---

## 💡 Key Insights

* Identified **top-performing companies and services**
* Tracked **pending dues vs received payments**
* Observed **regional sales trends and variations**
* Highlighted **quarterly performance patterns**
